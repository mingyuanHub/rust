# Rust Learning

Personal Rust learning projects, covering core language features from basics to advanced topics.

## Projects

| Project | Topics |
|---------|--------|
| `variables/` | Variables, mutability, shadowing, data types, constants |
| `func/` | Functions, control flow, expressions vs statements |
| `guessgame/` | User input, `match`, `loop`, random numbers |
| `ownership/` | Ownership, borrowing, references, dangling pointers |
| `slices/` | String slices, array slices, slice safety |
| `strings-my/` | `String` vs `&str`, string methods, concatenation |
| `vec-my/` | `Vec`, arrays, slices, generics, iterators |
| `structs/` | Structs, methods, associated functions, `impl` |
| `enum-my/` | Enums, `Option`, pattern matching |
| `crate-my/` | Modules, crates, visibility |
| `axum-api/` | HTTP server with Axum, async/await, Tokio |

## Docs

Learning notes organized by topic and difficulty — see [`_docs/`](./_docs/README.md).

## Requirements

- Rust 1.85+ (`rustup update`)
- Cargo (included with Rust)

## Run

```bash
cd <project>
cargo run
```
